Introduction

This is the simple page that shows a number of bootstrap components and the styles they provide to enhance plain HTML content - while also supporting a responsive display (screen resizing).

Firstly, at the top of the page, within a <header>... </header> section, is a navigation list using the Bootstrap <nav>...</nav> component. Next is the <body>...</body> section that contains a banner (a title of the page and leading sentence, all with a blue background), and the content of the page. The content contains some pretty pictures of butterflies (using different layouts), as well as a bootstrap formatted table with content. The final section is a <footer>...</footer>.
